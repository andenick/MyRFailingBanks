# Inputs Directory

This folder follows Druck compliance standards for input file organization.

## Subdirectories

| Folder | Purpose |
|--------|---------|
| `PDFs/` | Academic papers, regulatory documents, reference materials |
| `Excel/` | Source Excel files (e.g., FDIC data, regulatory reports) |
| `Documents/` | Word documents, text files, specifications |
| `Images/` | Diagrams, screenshots, visual references |
| `Data/` | Raw data files (CSV, RDS, DTA, etc.) |

## Data Sources for This Project

The analysis uses data from:

1. **FDIC Call Reports** - Quarterly regulatory filings
2. **FDIC Failures Database** - Official failure dates and resolution types
3. **Federal Reserve Economic Data (FRED)** - Macroeconomic variables

See `data/README.md` for detailed information on data sources and processing.

## Note

This is a GitHub repository focused on code and analysis outputs. Raw input data files are not included due to size and licensing restrictions. See `data/README.md` for instructions on obtaining the source data.
