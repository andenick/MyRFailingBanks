## CONTRIBUTING TO DOCUMENTATION

### Standards
1. All .md files must have:
   - Status header (Active/Archived/Superseded)
   - Last Updated date
   - Table of contents (if >100 lines)
   - See Also section

2. Naming conventions:
   - Descriptive titles in CAPS_WITH_UNDERSCORES.md
   - Version-specific: V8_0_*.md
   - Date-stamped: *_2025-11-15.md
   - Consolidated: *_CONSOLIDATED.md

3. Organization:
   - Root: Active master docs only
   - Documentation/Archive/: Historical docs
   - Documentation/Markdown/: Specialized docs
