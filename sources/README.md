# Source Data

**Status**: User must obtain

## Required Datasets

1. **OCC Historical Call Reports** (1863-1947)
   - File: `call-reports-historical.dta`
   - Size: ~232 MB
   - Source: Office of the Comptroller of the Currency

2. **FFIEC Modern Call Reports** (1959-2023)
   - File: `call-reports-modern.dta`
   - Size: ~343 MB
   - Source: Federal Financial Institutions Examination Council

3. **Receivership Records**
   - Files: `receivership_records.dta`, `deposits_before_failure_historical.dta`
   - Size: ~45 MB total
   - Source: OCC/FDIC

4. **Macro Data (GFD)**
   - Files: `GDP_annual.xlsx`, `CPI_monthly.xlsx`, `bond_yields.xlsx`, `stock_prices.xlsx`
   - Size: ~21 MB total
   - Source: Global Financial Data

**Total Required**: 3.2 GB
