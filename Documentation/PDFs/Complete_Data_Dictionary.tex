\subsection{temp\_reg\_data.dta}

\textbf{Description}: Regression-ready dataset with failure indicators and control variables for econometric analysis.

\textbf{Coverage}: 2,120,000 observations (subset of combined-data with complete cases)

\textbf{Key Filters Applied}:
\begin{itemize}
    \item Dropped observations after bank failure (quarters\_to\_failure > 0)
    \item Dropped if income\_ratio missing and year > 1941
    \item Dropped de novo banks (age < 3 years)
    \item Dropped if missing lagged variables
\end{itemize}

\textbf{Variables}: Same as combined-data.dta, but with complete cases only

\textbf{Purpose}: Used in all regression analyses (logit, OLS, event studies)

\section{Variable Construction Notes}

\subsection{Failure Indicators}

\subsubsection{Backward-Looking (failed\_bank)}

Permanent indicator: Bank has failed at some point in the sample period.

\textbf{Construction}:
\begin{verbatim}
failed_bank = 1 if fail_day is not missing
            = 0 otherwise
\end{verbatim}

\subsubsection{Forward-Looking (F1, F3, F5 failure)}

Time-varying indicators: Bank will fail within N quarters.

\textbf{Construction}:
\begin{verbatim}
F1_failure = 1 if quarters_to_failure <= 1 and quarters_to_failure > 0
           = 0 otherwise

F3_failure = 1 if quarters_to_failure <= 3 and quarters_to_failure > 0
           = 0 otherwise

F5_failure = 1 if quarters_to_failure <= 5 and quarters_to_failure > 0
           = 0 otherwise
\end{verbatim}

\subsection{Financial Ratios}

All ratios are expressed as percentages (0-100 scale).

\textbf{Equity Ratio}:
\begin{verbatim}
equity_ratio = (equity / assets) * 100
\end{verbatim}

\textbf{Loan Ratio}:
\begin{verbatim}
loan_ratio = (loans / assets) * 100
\end{verbatim}

\textbf{Liquid Ratio}:
\begin{verbatim}
liquid_ratio = (liquid / assets) * 100
where liquid = cash + securities (modern)
            or = specie + legal_tender + ... (historical)
\end{verbatim}

\textbf{Income Ratio}:
\begin{verbatim}
income_ratio = (net_income / assets) * 100
\end{verbatim}

\subsection{Lagged Variables}

Created using panel data lag operators (by bank ID, sorted by date).

\textbf{Example}:
\begin{verbatim}
L_equity_ratio = equity_ratio[t-1] for same bank
\end{verbatim}

Missing if:
\begin{itemize}
    \item First observation for bank
    \item Gap in reporting
\end{itemize}

\subsection{Bank Runs}

\textbf{Definition}: Deposit outflow > 15\% in single quarter

\textbf{Construction}:
\begin{verbatim}
deposits_growth = (deposits[t] - deposits[t-1]) / deposits[t-1] * 100

run = 1 if deposits_growth < -15
    = 0 otherwise
    = NA if deposits[t-1] missing
\end{verbatim}

\subsection{Asset Growth Quintiles}

\textbf{Growth Rate}:
\begin{verbatim}
growth = (log_assets[t] - log_assets[t-3]) / 3 * 100
\end{verbatim}

\textbf{Quintiles}: Calculated annually within era-group using xtile function:
\begin{itemize}
    \item growth\_cat = 1: Slowest growth
    \item growth\_cat = 2-4: Middle growth
    \item growth\_cat = 5: Fastest growth
\end{itemize}

\subsection{Known Data Limitations}

\subsubsection{Historical Era (1865-1958)}
\begin{itemize}
    \item Annual reporting (quarterly interpolated)
    \item Smaller banks sometimes missing
    \item Income data sparse before 1920
    \item Limited detail on asset composition
\end{itemize}

\subsubsection{Modern Era (1959-2024)}
\begin{itemize}
    \item Definitional changes in 1980s (thrifts)
    \item Merger tracking incomplete
    \item Small bank exemptions from certain reports
\end{itemize}

\section{Citation and Usage}

\subsection{Data Sources}

\textbf{Historical Call Reports}:
\begin{itemize}
    \item Office of the Comptroller of the Currency (OCC)
    \item Annual reports of national banks (1865-1958)
    \item Digitized by authors
\end{itemize}

\textbf{Modern Call Reports}:
\begin{itemize}
    \item Federal Deposit Insurance Corporation (FDIC)
    \item Quarterly call reports (1959-2024)
    \item Accessed via FDIC API
\end{itemize}

\textbf{Macro Data}:
\begin{itemize}
    \item Barro-Ursua Macroeconomic Dataset
    \item Bureau of Economic Analysis (BEA)
    \item Global Financial Data (GFD)
\end{itemize}

\subsection{Recommended Citation}

\textbf{Paper}:
\begin{quote}
Correia, Sergio, Stephan Luck, and Emil Verner. 2025. ``Failing Banks.'' \textit{Quarterly Journal of Economics} (Forthcoming).
\end{quote}

\textbf{R Replication Data}:
\begin{quote}
R Replication Package for ``Failing Banks'' (2025). Data dictionary and documentation.
\end{quote}

\subsection{Usage Notes}

\begin{enumerate}
    \item All financial variables are in nominal USD
    \item Ratios are percentages (0-100 scale)
    \item Missing values coded as NA (not 999 or -1)
    \item Panel data sorted by: bank ID, year, quarter
    \item Lagged variables use t-1 (one period lag)
\end{enumerate}

\subsection{Variable Name Conventions}

\begin{itemize}
    \item \textbf{L\_*}: Lagged variable (t-1)
    \item \textbf{F[N]\_failure}: Fails within N quarters
    \item \textbf{*\_ratio}: Financial ratio (percent)
    \item \textbf{log\_*}: Natural logarithm
    \item \textbf{*\_growth}: Growth rate (percent)
    \item \textbf{*\_cat}: Categorical variable
\end{itemize}

\subsection{Key Thresholds}

\begin{itemize}
    \item \textbf{Bank run}: Deposit outflow > 15\%
    \item \textbf{De novo}: Age < 3 years
    \item \textbf{Small bank}: Assets < \$100M (2024 USD)
    \item \textbf{Large bank}: Assets > \$10B (2024 USD)
\end{itemize}
