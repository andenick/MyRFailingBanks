\section{What This Package Does}

This R replication package reproduces all analyses from:

\begin{quote}
\textbf{Correia, Sergio, Stephan Luck, and Emil Verner. 2025.}\\
``Failing Banks.'' \textit{Quarterly Journal of Economics} (Forthcoming).
\end{quote}

\subsection{Key Findings Replicated}

\begin{enumerate}
    \item \textbf{Shrinking banks fail much more}: 6.72\% vs 4.19\% (60\% higher)
    \item \textbf{Low equity predicts failure}: Coefficient $-4.52$ ($p<0.001$)
    \item \textbf{Patterns stable 160 years}: Same pre- and post-FDIC
    \item \textbf{Runs amplify risk}: 15-20\% vs 3-5\% (3-4$\times$ multiplier)
    \item \textbf{Recovery dismal}: 99.94\% depositor loss without insurance
\end{enumerate}
